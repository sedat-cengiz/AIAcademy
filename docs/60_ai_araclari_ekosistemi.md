# AI Araçları Ekosistemi

## Kimin için?

- AI dünyasını daha geniş bir perspektiften görmek isteyen herkes
- Hangi aracın hangi iş için uygun olduğunu merak edenler

## Öğrenme hedefleri

- Güncel AI araç ekosistemini tanımak
- Farklı araçların güçlü ve zayıf yönlerini bilmek
- İş ihtiyacınıza uygun aracı seçebilmek
- AI araçlarını sınıflandırabilmek

## AI araçları haritası

### 1. Metin tabanlı AI asistanları (LLM sohbet)

En yaygın kullanılan araç kategorisi. Metin girdisi alır, metin çıktısı üretir.

**Claude (Anthropic)**
- Güçlü yönler: Uzun bağlam, dürüstlük, analitik görevler, Türkçe
- Öne çıkan: 200K+ token bağlam penceresi, güvenlik odaklı tasarım
- Kullanım: Sohbet arayüzü, API, Claude Code (IDE)

**ChatGPT (OpenAI)**
- Güçlü yönler: Geniş bilgi tabanı, eklenti ekosistemi, görsel anlama
- Öne çıkan: GPT-4o çoklu modalite, özel GPT'ler oluşturma
- Kullanım: Sohbet, API, mobil uygulama

**Gemini (Google)**
- Güçlü yönler: Google ekosistemi entegrasyonu, çoklu modalite
- Öne çıkan: Google Workspace ile doğal entegrasyon
- Kullanım: Web, Google ürünleri içinde

**Copilot (Microsoft)**
- Güçlü yönler: Microsoft 365 entegrasyonu, kurumsal güvenlik
- Öne çıkan: Word, Excel, PowerPoint, Teams ile entegre
- Kullanım: Microsoft ürünleri içinde

### 2. Kod geliştirme araçları

**Claude Code (Anthropic)**
- Terminal tabanlı AI geliştirme aracı
- Proje bağlamını anlama, dosya düzenleme, test yazma

**GitHub Copilot**
- Kod tamamlama ve önerme
- VS Code, JetBrains IDE'leri ile entegre

**Cursor**
- AI-first kod editörü
- Claude, GPT modelleri ile çalışır
- Doğal dil ile kod düzenleme

### 3. Görsel üretim araçları

**DALL-E (OpenAI)** — Metinden görsel üretimi
**Midjourney** — Yüksek kaliteli sanatsal görseller
**Stable Diffusion** — Açık kaynak, yerel çalıştırılabilir

### 4. Ses ve video araçları

**Whisper (OpenAI)** — Konuşmayı yazıya dökme
**ElevenLabs** — Metin-ses dönüşümü
**Descript** — AI destekli video/ses düzenleme

### 5. Özelleşmiş iş araçları

**Notion AI** — Not ve doküman yönetiminde AI
**Otter.ai** — Toplantı kayıt ve özet
**Grammarly** — Yazım ve dil bilgisi kontrolü
**Jasper** — Pazarlama içeriği üretimi

## Araç seçim rehberi

### "Hangi aracı ne için kullanmalıyım?"

| İhtiyaç | Önerilen araç(lar) |
|---------|---------------------|
| Metin özetleme, analiz | Claude, ChatGPT |
| E-posta ve doküman yazımı | Claude, Copilot (M365) |
| Kod yazma ve inceleme | Claude Code, GitHub Copilot, Cursor |
| Excel/veri analizi | Copilot (Excel), ChatGPT |
| Sunum hazırlama | Copilot (PowerPoint), Claude |
| Toplantı özeti | Otter.ai, Copilot (Teams) |
| Görsel üretimi | DALL-E, Midjourney |
| Müşteri destek otomasyon | Claude API, ChatGPT API |

### Seçim kriterleri

1. **Güvenlik**: Kurumsal veri politikanıza uygun mu?
2. **Entegrasyon**: Mevcut araçlarınızla çalışıyor mu?
3. **Maliyet**: Bütçenize uygun mu?
4. **Yetenek**: İhtiyacınızı karşılıyor mu?
5. **Türkçe destek**: Yeterli düzeyde Türkçe anlıyor ve üretiyor mu?

## Yerel (on-premise) modeller

### Nedir?

AI modellerini kendi sunucunuzda çalıştırmak. Verileriniz dışarıya çıkmaz.

### Ne zaman düşünülmeli?

- Çok hassas verilerle çalışıyorsanız
- İnternet bağımsız çalışma gerekiyorsa
- Maliyet kontrolü istiyorsanız (yüksek hacim)
- Özelleştirilmiş model gerekiyorsa

### Popüler yerel modeller

- **Llama (Meta)** — Açık kaynak, güçlü genel amaçlı model
- **Mistral** — Hafif ve hızlı Avrupa modeli
- **Phi (Microsoft)** — Küçük ama yetenekli modeller

### Dikkat edilmesi gerekenler

- Donanım gereksinimi yüksek (GPU)
- Kurulum ve bakım maliyeti
- Bulut modellere göre genellikle daha düşük performans
- Güncelleme ve bakım sorumluluğu sizde
